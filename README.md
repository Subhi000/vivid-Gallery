# vivid-Gallery
VividGallery is a modern and responsive image gallery web application built with HTML, CSS, and JavaScript. It features category-based browsing, global image search powered by the Pexels API, lightbox viewing, dark/light mode, favorites, and an immersive user experience designed for seamless image exploration.
# 🌟 VividGallery

VividGallery is a modern and interactive image gallery web application built using **HTML, CSS, and JavaScript**. It provides users with a visually engaging platform to explore curated image collections and search for high-quality images from around the world using the **Pexels API**.

## 🚀 Features

- 🔍 Global image search powered by the Pexels API
- 🖼️ Browse images through multiple categories
- ❤️ Save favorite images for quick access
- 🌙 Dark Mode / ☀️ Light Mode support
- 🔎 Lightbox view with next and previous navigation
- ⌨️ Keyboard shortcuts for seamless navigation
- 📥 Download images directly from the gallery
- 📱 Fully responsive design for desktop, tablet, and mobile devices
- ⚡ Lazy loading for improved performance
- 🎨 Smooth animations and interactive user interface
- 📌 Pinterest-style masonry layout
- 🏆 Achievement badges and hidden Easter eggs for enhanced user engagement

## ⚙️ Installation

1. Clone the repository:

```bash
git clone https://github.com/your-username/VividGallery.git
```

2. Navigate to the project folder:

```bash
cd VividGallery
```

3. Open `index.html` in your browser.

## 🔑 API Setup

1. Create a free account on Pexels.
2. Generate your API key from the Pexels API dashboard.
3. Create a `config.js` file:

```javascript
const PEXELS_API_KEY = "YOUR_API_KEY_HERE";
```

4. Ensure `config.js` is included in your `.gitignore` file.
